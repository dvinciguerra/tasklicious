=head2 Tasklicious - Mojolicious Task WebApp

This is a simple task application build using C<Mojolicious>


=head3 Technologies used

=over

=item Perl 5.10+

=item Mojolicious

=item DBIx::Class

=item MySQL

=back


=head3 Using this app

First clone this git repo:

    $ git clone https://github.com/dvinciguerra/tasklicious.git

Now you can run morbo to start local server...

    $ morbo index.pl
